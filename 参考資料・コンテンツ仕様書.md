## 2. セルフチェック設問一覧

> **注意事項**
> 以下の設問は国際標準尺度に基づく参考情報です。本アプリのセルフチェックは医療診断を目的とするものではありません。スコアが高い場合は、専門家への相談を促してください。

### 2.1 PHQ-9（Patient Health Questionnaire-9）— うつ症状

**出典：** Kroenke K, et al., 2001  
**回答方式：** 4段階評価（過去2週間について）  
**所要時間：** 約2〜3分  

**回答選択肢**
| 選択肢 | スコア |
|--------|--------|
| 全くない | 0 |
| 数日（週に1〜2日） | 1 |
| 半分以上（週に3〜4日） | 2 |
| ほぼ毎日 | 3 |

**設問一覧（過去2週間のあいだ、どのくらいの頻度で次のことがありましたか？）**

| Q | 設問文 | 測定内容 |
|---|--------|----------|
| Q1 | 物事に対してほとんど興味がない、または楽しめない | 興味の喪失 |
| Q2 | 気分が沈んでいる、または絶望的な気持ちになる | 抑うつ気分 |
| Q3 | 寝つきが悪い、途中で目が覚める、または寝すぎる | 睡眠障害 |
| Q4 | 疲れた感じがする、または気力がない | 疲労感 |
| Q5 | あまり食欲がない、または食べすぎる | 食欲変化 |
| Q6 | 自分はダメな人間だ、または自分を責める気持ちになる | 自己評価の低下 |
| Q7 | 新聞を読む、またはテレビを見るなどのことに集中するのが難しい | 集中困難 |
| Q8 | 動きや話し方が遅くなっている、または落ち着かず、じっとしていられない | 精神運動変化 |
| Q9 | 死んだほうがましだ、または自分を何らかの方法で傷つけようと思ったことがある | 自傷・希死念慮 |

**判定基準（合計スコア 0〜27点）**
| スコア | 重症度 | アプリ内表示 | 対応 |
|--------|--------|------------|------|
| 0〜4 | 最小限 | 良好 | 継続的なケアを促す |
| 5〜9 | 軽度 | 注意 | セルフケアを推奨 |
| 10〜14 | 中等度 | 要注意 | 相談窓口の案内 |
| 15〜19 | 中等度〜重度 | 要注意 | 専門家受診を強く促す |
| 20〜27 | 重度 | 要注意（緊急） | 即時相談窓口へ誘導 |

> ⚠️ **Q9（希死念慮）で「数日以上」を選択した場合は、スコアに関わらず緊急サポートページへ誘導する。**

### 2.2 GAD-7（Generalized Anxiety Disorder-7）— 不安症状

**出典：** Spitzer RL, et al., 2006  
**回答方式：** 4段階評価（過去2週間について）  
**所要時間：** 約2〜3分  

**回答選択肢**（PHQ-9 と同じ 0〜3点）

**設問一覧**

| Q | 設問文 | 測定内容 |
|---|--------|----------|
| Q1 | 緊張感、不安感、またはびくびくした感じがある | 不安・緊張 |
| Q2 | 心配することを止められない、またはコントロールできない | 制御困難な心配 |
| Q3 | いろいろなことについてひどく心配する | 過剰な心配 |
| Q4 | リラックスするのが難しい | 弛緩困難 |
| Q5 | じっとしていられないくらい不安で落ち着かない | 落ち着かなさ |
| Q6 | 苛立ちやすくなったり、怒りっぽくなる | 易怒性 |
| Q7 | 何か恐ろしいことが起こるのではないかと恐れを感じる | 破局的思考 |

**判定基準（合計スコア 0〜21点）**
| スコア | 重症度 | アプリ内表示 |
|--------|--------|------------|
| 0〜4 | 最小限 | 良好 |
| 5〜9 | 軽度 | 注意 |
| 10〜14 | 中等度 | 要注意 |
| 15〜21 | 重度 | 要注意（専門家受診を推奨） |

### 2.3 PSQI 簡易版（Pittsburgh Sleep Quality Index）— 睡眠の質

**出典：** Buysse DJ, et al., 1989（簡易版として5設問に圧縮）  
**回答方式：** 設問ごとに異なる  
**所要時間：** 約2〜3分  

| Q | 設問文 | 回答形式 |
|---|--------|----------|
| Q1 | 普段の就寝時刻は何時ですか？ | 時刻入力（例：23:00） |
| Q2 | 寝るまでに通常どのくらいかかりますか？ | 4択（15分未満 / 15〜30分 / 31〜60分 / 60分以上） |
| Q3 | 普段の起床時刻は何時ですか？ | 時刻入力（例：7:00） |
| Q4 | 実際に眠れている時間は1日どのくらいですか？ | 4択（7時間以上 / 6〜7時間 / 5〜6時間 / 5時間未満） |
| Q5 | 過去1ヶ月の睡眠の質を全体的に評価してください | 4択（とても良い / まあ良い / 悪い / とても悪い） |

**判定基準（加重スコア換算後 0〜21点）**
| スコア | 評価 | アプリ内表示 |
|--------|------|------------|
| 0〜4 | 良質な睡眠 | 良好 |
| 5〜10 | やや問題あり | 注意 |
| 11〜21 | 睡眠の質に問題 | 要注意 |

### 2.4 レジリエンス指標（独自設問）— 精神的回復力

**測定目的：** ストレスへの対処能力・回復力を測定し、トレーニング前後の変化を追跡する  
**回答方式：** 5段階評価（「全くそう思わない」〜「とてもそう思う」）  
**実施頻度：** 月1回  
**所要時間：** 約2〜3分  

**回答選択肢**
| 選択肢 | スコア |
|--------|--------|
| 全くそう思わない | 1 |
| あまりそう思わない | 2 |
| どちらでもない | 3 |
| まあそう思う | 4 |
| とてもそう思う | 5 |

**設問一覧**

| Q | 設問文 | 測定するレジリエンス要素 |
|---|--------|----------------------|
| Q1 | つらいことがあっても、時間が経てば立ち直れると思う | 回復力・楽観性 |
| Q2 | 問題が起きたとき、解決のための行動が取れる | 問題対処能力 |
| Q3 | 誰かにサポートを求めることが苦手ではない | ソーシャルサポート活用 |
| Q4 | 失敗しても、そこから学べることがあると思える | 成長的思考 |
| Q5 | 嫌な気分の時でも、ポジティブな側面を見つけようとする | 認知的柔軟性 |
| Q6 | 強いストレスを感じても、何とかなると思える | 自己効力感 |
| Q7 | 今の自分の感情を客観的に観察できる | メタ認知能力 |
| Q8 | 大変な時期があっても、自分は成長できると信じている | 成長的思考 |

**判定基準（合計スコア 8〜40点）**
| スコア | 評価 |
|--------|------|
| 33〜40 | 高いレジリエンス |
| 24〜32 | 中程度のレジリエンス |
| 8〜23 | レジリエンスが低い（トレーニングで向上の余地あり）|

## 3. トレーニングコンテンツ詳細

> **プログラム設計の根拠**  
> 古川壽亮教授らの Healthy Campus Trial（Nature Medicine, 2025）で有効性が実証された5つの認知行動スキルを、1日5〜10分で学べるレッスン形式に分解したものです。介入期間は6週間を基本とし、各スキル週2〜3セッションの受講を推奨します。

---

### SK01：行動活性化（ポジティブ・レジリエンス）

**目的：** 気分と行動の好循環を作り出す  
**期待効果：** うつ気分の改善、意欲・エネルギーの回復  
**推奨対象：** PHQ-9スコアが5以上、やる気の低下を訴えるユーザー  
**総レッスン数：** 10レッスン  

| # | レッスンタイトル | 形式 | 所要時間 | 内容概要 |
|---|----------------|------|----------|----------|
| L1 | 行動活性化とは？ | 学習 | 5分 | 気分と行動の悪循環（うつの罠）について学ぶ。「まずやってみる」ことの重要性 |
| L2 | 気分と行動の関係 | 学習＋ワーク | 7分 | 「行動→気分」の因果を体感する。過去1週間で気分が上がった瞬間を記録 |
| L3 | 好きな活動をリストアップ | ワーク | 8分 | 以前楽しかったこと・してみたいことをリストアップ。活動記録表の作成 |
| L4 | 小さな行動計画を立てる | ワーク | 8分 | 来週取り組む小さな活動を1つ決める。スモールステップ化の技術 |
| L5 | 行動記録をつける | ワーク | 6分 | 1日の活動と気分（0〜10点）を記録。行動と気分の相関を確認 |
| L6 | 前週記録を振り返る | 振り返り | 7分 | 1週間の行動記録を振り返り、気分が上がった活動パターンを発見 |
| L7 | スケジューリングのコツ | 学習＋ワーク | 9分 | 楽しい活動を週間スケジュールに組み込む。達成可能な計画の立て方 |
| L8 | 活動を共有する | ワーク | 6分 | 信頼できる人と一緒に活動することの効果。活動の「社会化」 |
| L9 | 活動を維持するためのピットフォール | 学習 | 8分 | 継続を妨げるよくある落とし穴（「やる気待ち」「完璧主義」等）と対策 |
| L10 | まとめと実践プラン | 振り返り | 10分 | 6週間後の行動目標を設定。マイ行動活性化プランの完成 |

**ワークのポイント**
- 毎日の活動記録（活動内容・気分点数）
- 週1回の「気分が上がった活動TOP3」振り返り
- 翌週の活動計画（少なくとも3つ）

### SK02：認知再構成（メタ・レジリエンス）

**目的：** ネガティブな思考パターンに気づき、柔軟でバランスのとれた見方に変える  
**期待効果：** 不安・ストレス・自己批判の軽減  
**推奨対象：** GAD-7スコアが5以上、ネガティブ思考が強いユーザー  
**総レッスン数：** 12レッスン  

| # | レッスンタイトル | 形式 | 所要時間 | 内容概要 |
|---|----------------|------|----------|----------|
| L1 | 思考と感情の関係 | 学習 | 5分 | 「出来事→思考→感情」のABCモデルを学ぶ |
| L2 | 自動思考に気づく | 学習＋ワーク | 7分 | 頭に浮かぶ自動的な考え（自動思考）を捕まえる練習 |
| L3 | 認知の歪みを知る① | 学習 | 8分 | 「全か無か思考」「過度な一般化」「心の読みすぎ」の解説 |
| L4 | 認知の歪みを知る② | 学習 | 8分 | 「破局化」「感情的決めつけ」「べき思考」「レッテル貼り」の解説 |
| L5 | 自分の歪みパターンを見つける | ワーク | 7分 | 過去のストレス場面を振り返り、自分がよく使う歪みを特定 |
| L6 | 証拠を集める | ワーク | 9分 | ネガティブ思考を「法廷での証言」のように客観的に検討する |
| L7 | 別の見方を探す | ワーク | 8分 | 「もし友人が同じ状況にいたら何と言うか」で思考を書き換える |
| L8 | バランスのとれた考えを作る | ワーク | 9分 | 偏りのない「バランス思考」を文章化する練習 |
| L9 | コアビリーフを探る | 学習＋ワーク | 10分 | 思考の根底にある深い信念（コアビリーフ）を探り、柔軟化する |
| L10 | 思考記録表を使いこなす | ワーク | 8分 | 状況・感情・自動思考・根拠・バランス思考を記録する表の活用 |
| L11 | セルフ・コンパッション | 学習＋ワーク | 8分 | 自分への思いやり（自己批判の緩和）を育てる |
| L12 | まとめと振り返り | 振り返り | 10分 | 認知再構成スキルの定着確認。今後の活用プランを作成 |

### SK03：問題解決（問題解決・レジリエンス）

**目的：** 問題を整理し、実行可能な対処法を系統的に見つける  
**期待効果：** 課題遂行力の向上、ストレス対処能力の強化  
**推奨対象：** 就活・学業・対人関係など具体的な悩みを持つユーザー  
**総レッスン数：** 8レッスン  

| # | レッスンタイトル | 形式 | 所要時間 | 内容概要 |
|---|----------------|------|----------|----------|
| L1 | 問題解決法とは？ | 学習 | 5分 | 「問題を解こうとしないこと」がうつを悪化させるメカニズムを学ぶ |
| L2 | 問題を明確に定義する | ワーク | 7分 | モヤモヤした悩みを「具体的で解決可能な問題」に言い換える技術 |
| L3 | ブレインストーミング | ワーク | 8分 | 解決策をジャッジなしに列挙する。良い案も悪い案もとにかく出す |
| L4 | 解決策を評価・選択する | ワーク | 8分 | 各解決策のメリット・デメリット、実行可能性を評価して選択 |
| L5 | アクションプランを立てる | ワーク | 9分 | 選んだ解決策を「いつ・どこで・何をする」の形で具体化 |
| L6 | 実行して振り返る | 振り返り | 7分 | 試みた解決策の結果を振り返り、次のステップを考える |
| L7 | 「解決できない問題」との向き合い方 | 学習＋ワーク | 9分 | 自分でコントロールできる部分とできない部分を区別する |
| L8 | まとめ | 振り返り | 7分 | 問題解決の6ステップを習得。日常への応用プランを作成 |

**問題解決の6ステップ（アプリ内で常時参照可能）**
1. 問題の明確化（漠然とした悩み→具体的な問題）
2. 目標の設定（どうなれば解決したか）
3. 解決策の列挙（ブレインストーミング）
4. 解決策の評価と選択
5. アクションプランの作成と実行
6. 結果の評価（うまくいかなければ2に戻る）

### SK04：アサーション（コミュ・レジリエンス）

**目的：** 自分も相手も尊重した適切な自己表現・コミュニケーション力を高める  
**期待効果：** 対人関係ストレスの軽減、自己主張力の向上  
**推奨対象：** 人間関係の悩み・断れない・言いたいことが言えないユーザー  
**総レッスン数：** 10レッスン  

| # | レッスンタイトル | 形式 | 所要時間 | 内容概要 |
|---|----------------|------|----------|----------|
| L1 | アサーションとは？ | 学習 | 5分 | 「攻撃的・受け身的・アサーティブ」の3スタイルの違いを学ぶ |
| L2 | 自分のコミュニケーションスタイルを知る | ワーク | 7分 | チェックリストで自分の傾向（攻撃型/受動型）を把握 |
| L3 | 断る権利 | 学習＋ワーク | 8分 | 「No」と言うことへの罪悪感を解消する。権利としての断り方 |
| L4 | DEスクリプト法を学ぶ | 学習 | 7分 | Describe（描写）・Express（表現）・Specify（提案）・Consequence（結果）の4ステップ |
| L5 | 依頼・断りの練習 | ワーク | 9分 | シナリオを使った「お願い」「断り」のロールプレイ的ワーク |

### SK05：睡眠行動療法（睡眠・レジリエンス）

**目的：** 睡眠の質を改善する行動習慣（CBT-I）を身につける  
**期待効果：** 睡眠の質向上、疲労回復、昼間のパフォーマンス向上  
**推奨対象：** PSQI簡易版スコアが5以上、睡眠の悩みを持つユーザー  
**総レッスン数：** 10レッスン
